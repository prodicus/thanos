Contributing
============

-  Conform to `PEP0008 <http://pep8.org>`__
-  Make sure your code passes **flake8** and **pep257**

.. code:: bash

    $ make flake8

For ``pep257``

.. code:: bash

    $ make pep257

- Running the tests

.. code:: bash
    $ make tests

-  Write meaningful commit messages
-  Rebase your commits to one commit when making a PR
